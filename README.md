| Command      | Description              |

| npm run dev  | Start development server |
| npm run test | Run tests                |

## Dev Server :computer:

After installing dependencies with command:

> npm install

You can run your development server with command:

> npm run dev

Then Application can be accessed with this URL:

> http://localhost:2137/
